# Polysmash
Polysmash is a 3D arena game built on **Three.js**.
This game has some unnescesairy graphics, and *may* crash your browser tab.

# Controls
**W** - Strafe forward  
**S** - Strafe backward  
**D** - Strafe right  
**A** - Strafe left  
**Shift** - Sprint  
**V** - Toggle 3rd person view  
**Esc** - Pause game and show your cursor
